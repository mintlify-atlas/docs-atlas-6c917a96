A captivating lamp lighting effect with conic gradients, spotlights, and glowing lines for an immersive visual experience. This component creates a dramatic lighting effect perfect for hero sections.

## Props

| Prop Name  | Type     | Default | Description                                    |
| ---------- | -------- | ------- | ---------------------------------------------- |
| `delay`    | `number` | `0.5`   | Delay before the animation starts, in seconds. |
| `duration` | `number` | `0.8`   | Duration of the animation, in seconds.         |
| `class`    | `string` | `""`    | Additional CSS classes for custom styling.     |
