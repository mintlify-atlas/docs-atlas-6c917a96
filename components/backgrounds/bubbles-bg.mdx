An animated background with floating bubbles built using Three.js for 3D rendering and animations.

## Dependencies

```bash
npm install three
npm install -D @types/three
```

## Props

| Prop Name | Type     | Default | Description                                                     |
| --------- | -------- | ------- | --------------------------------------------------------------- |
| `blur`    | `number` | `0`     | Amount of blur to apply to the background, specified in pixels. |
