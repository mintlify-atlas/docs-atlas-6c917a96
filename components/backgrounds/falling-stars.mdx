A stunning animated starfield background with glowing and sharp trail effects. This component creates a beautiful 3D starfield simulation using canvas animations.

## Props

| Prop Name | Type     | Default  | Description                                 |
| --------- | -------- | -------- | ------------------------------------------- |
| `color`   | `string` | `"#FFF"` | Color of the stars in the starfield.        |
| `count`   | `number` | `200`    | Number of stars displayed in the animation. |
