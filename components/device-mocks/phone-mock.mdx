A detailed iPhone 15 Pro mockup component that displays your mobile app or responsive website design in a realistic device frame. Perfect for app showcases and mobile-first presentations.

## Installation

<Steps>
  <Step title="Install dependencies">
    This component works with Tailwind CSS. No additional dependencies required.
  </Step>
  
  <Step title="Copy component code">
    Add the `iPhone15ProMockup.vue` component to your project.
  </Step>
</Steps>

## Usage

### Basic Usage with Screenshot

```vue
<template>
  <iPhone15ProMockup src="/screenshots/mobile-app.png" />
</template>

<script setup>
import iPhone15ProMockup from '@/components/inspira/ui/iPhone15ProMockup.vue'
</script>
```

### Custom Dimensions

```vue
<template>
  <iPhone15ProMockup 
    src="/screenshots/app.png"
    :width="500"
    :height="1020"
  />
</template>
```

### With Slot Content

```vue
<template>
  <iPhone15ProMockup>
    <div class="h-full bg-white p-6">
      <div class="flex flex-col h-full">
        <h1 class="text-2xl font-bold mb-4">My App</h1>
        <div class="flex-1 overflow-y-auto">
          <!-- Your app UI here -->
        </div>
      </div>
    </div>
  </iPhone15ProMockup>
</template>
```

### Multiple Mockups in Grid

```vue
<template>
  <div class="grid grid-cols-1 md:grid-cols-3 gap-8 items-center justify-items-center">
    <iPhone15ProMockup src="/screens/onboarding.png" />
    <iPhone15ProMockup src="/screens/home.png" />
    <iPhone15ProMockup src="/screens/profile.png" />
  </div>

```

### Responsive Container

```vue
<template>
  <div class="max-w-sm mx-auto">
    <iPhone15ProMockup 
      src="/screenshots/responsive-design.png"
    />
  </div>
</template>
```

## Props

<ResponseField name="src" type="string" default="null">
  URL of the image to display inside the mockup. Use a screenshot of your mobile app or responsive website.
</ResponseField>

<ResponseField name="width" type="number" default="433">
  Width of the mockup SVG in pixels. The default maintains the iPhone 15 Pro's aspect ratio.
</ResponseField>

<ResponseField name="height" type="number" default="882">
  Height of the mockup SVG in pixels. The default maintains the iPhone 15 Pro's aspect ratio.
</ResponseField>

## Features

- **iPhone 15 Pro Design**: Accurate representation of the latest iPhone model
- **Authentic Details**: Includes Dynamic Island, button placement, and bezels
- **Flexible Content**: Support for images or custom slot content
- **Customizable Size**: Adjust dimensions while maintaining aspect ratio
- **SVG-based**: Crisp rendering at any scale
- **Lightweight**: No external image dependencies

## Use Cases

- **App Store Previews**: Showcase mobile applications
- **Portfolio**: Display mobile design projects  
- **Landing Pages**: Demonstrate app features
- **Documentation**: Show mobile UI examples
- **Presentations**: Create professional mockups for pitches
- **App Walkthroughs**: Display onboarding flows

## Tips

- For best results, use screenshots with dimensions matching the iPhone 15 Pro screen ratio (roughly 1:2)
- Combine with other components like carousels to show multiple screens
- Use the slot for live, interactive content instead of static screenshots
- The default size works well for most layouts, but adjust for your needs
