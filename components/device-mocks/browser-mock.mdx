A high-quality Safari browser mockup component that displays your web content in a realistic Safari window. Perfect for landing pages, portfolios, and product showcases.

## Installation

<Steps>
  <Step title="Install dependencies">
    This component works with Tailwind CSS. No additional dependencies required.
  </Step>
  
  <Step title="Copy component code">
    Add the `SafariMockup.vue` component to your project.
  </Step>
</Steps>

## Usage

### Basic Usage with Image

```vue
<template>
  <SafariMockup 
    url="https://example.com"
    src="/screenshots/website-preview.png"
  />
</template>

<script setup>
import SafariMockup from '@/components/inspira/ui/SafariMockup.vue'
</script>
```

### Custom Dimensions

```vue
<template>
  <SafariMockup 
    url="https://myapp.com"
    src="/screenshots/app.png"
    :width="1400"
    :height="900"
  />
</template>
```

### With Slot Content

```vue
<template>
  <SafariMockup url="https://mywebsite.com">
    <div class="p-8 bg-gradient-to-br from-blue-500 to-purple-600">
      <h1 class="text-4xl text-white font-bold">Live Content</h1>
      <p class="text-white mt-4">Any custom content can go here</p>
    </div>
  </SafariMockup>
</template>
```

### Responsive Sizing

```vue
<template>
  <div class="w-full max-w-4xl mx-auto">
    <SafariMockup 
      url="https://responsive-site.com"
      src="/screenshots/responsive.png"
    />
  </div>
</template>
```

## Props

<ResponseField name="url" type="string" default="null">
  URL displayed in the mockup's address bar. Shows the website address at the top of the browser window.
</ResponseField>

<ResponseField name="src" type="string" default="null">
  URL of the image to display inside the mockup. Use this to show a screenshot of your website or app.
</ResponseField>

<ResponseField name="width" type="number" default="1203">
  Width of the mockup SVG in pixels. Adjust based on your layout needs.
</ResponseField>

<ResponseField name="height" type="number" default="753">
  Height of the mockup SVG in pixels. Maintains aspect ratio with width.
</ResponseField>

## Features

- **Pixel-Perfect Design**: Authentic Safari browser appearance
- **Customizable Dimensions**: Adjust size to fit your layout
- **URL Display**: Show the website address in the address bar
- **Flexible Content**: Use images or slot content
- **Responsive**: Works with responsive layouts
- **SVG-based**: Sharp on all screen resolutions
