## Usage

An iPhone mockup component that displays content inside an SVG representation of an iPhone.

<CodeGroup>
```vue Vue
<iPhone15ProMockup
  :width="433"
  :height="882"
  src="/path/to/your/image.png"
/>
```
</CodeGroup>

## Props

<ParamField path="width" type="number" default="433">
  Width of the mockup SVG in pixels.
</ParamField>

<ParamField path="height" type="number" default="882">
  Height of the mockup SVG in pixels.
</ParamField>

<ParamField path="src" type="string" default="null">
  URL of the image to display inside the mockup.
</ParamField>
