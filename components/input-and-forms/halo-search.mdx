## Overview

A futuristic search input component featuring multiple glowing rings and subtle animated halos. Perfect for creating a vibrant, immersive UI experience with a modern, sci-fi aesthetic.

## Usage

```vue
<template>
  <HaloSearch class="custom-search" />
</template>
```

## Props

| Prop Name | Type     | Default | Description                                |
| --------- | -------- | ------- | ------------------------------------------ |
| `class`   | `string` | `""`    | Additional CSS classes for root container. |

## Features

- Multiple glowing rings with animated halo effects
- Smooth, ambient lighting animations
- Futuristic UI design
- Fully customizable styling via CSS classes
