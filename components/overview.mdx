Inspira UI provides a curated collection of elegant components that you can easily integrate into your Vue applications. This is not a traditional component library - instead, you simply copy the code you need and customize it to fit your project.

## Copy-Paste Approach

Unlike traditional npm packages, Inspira UI uses a **copy-paste approach**:

1. Browse the component categories below to find what you need
2. Visit the component page to see live demos and code
3. Copy the component code directly into your project
4. Customize and modify the code to match your design requirements

The code is yours to use and modify as you like - no dependencies, no bloat, just clean Vue components.
