## Overview

Scroll Island is a component that displays scroll progress with animated number transitions and an expandable interface for additional content.

## Usage

```vue
<template>
  <ScrollIsland
    title="Progress"
    :height="44"
    class="custom-class"
  >
    <!-- Additional content goes here -->
  </ScrollIsland>
</template>
```

## API Reference

### Props

| Prop Name | Type     | Default      | Description                                     |
| --------- | -------- | ------------ | ----------------------------------------------- |
| `class`   | `string` | `""`         | Additional CSS classes for custom styling.      |
| `title`   | `string` | `"Progress"` | Title displayed in the header of the component. |
| `height`  | `string` | `44`         | Height of the component.                        |
