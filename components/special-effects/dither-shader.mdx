## Overview

The Dither Shader component applies real-time ordered dithering effects to images, creating pixel art and retro aesthetics. It supports multiple dithering modes, color processing options, and customizable parameters.

## Usage

```vue
<DitherShader
  src="/path/to/image.jpg"
  :gridSize="4"
  ditherMode="bayer"
  colorMode="duotone"
  primaryColor="#000000"
  secondaryColor="#ffffff"
  class="w-full h-96"
/>
```

## Props

<ResponseField name="src" type="string" required>
  Source image URL.
</ResponseField>

<ResponseField name="gridSize" type="number" default="4">
  Size of the dithering grid cells.
</ResponseField>

<ResponseField name="ditherMode" type="'bayer' | 'halftone' | 'noise' | 'crosshatch'" default="bayer">
  Type of dithering pattern.
</ResponseField>

<ResponseField name="colorMode" type="'original' | 'grayscale' | 'duotone' | 'custom'" default="original">
  Color processing mode.
</ResponseField>

<ResponseField name="invert" type="boolean" default="false">
  Invert the dithered output colors.
</ResponseField>

<ResponseField name="pixelRatio" type="number" default="1">
  Pixelation multiplier (1 = no pixelation, higher = more pixelated).
</ResponseField>

<ResponseField name="primaryColor" type="string" default="#000000">
  Primary color for duotone mode.
</ResponseField>

<ResponseField name="secondaryColor" type="string" default="#ffffff">
  Secondary color for duotone mode.
</ResponseField>

<ResponseField name="customPalette" type="string[]" default="['#000000', '#ffffff']">
  Custom color palette array for custom mode.
</ResponseField>

<ResponseField name="brightness" type="number" default="0">
  Brightness adjustment (-1 to 1).
</ResponseField>

<ResponseField name="contrast" type="number" default="1">
  Contrast adjustment (0 to 2, 1 = normal).
</ResponseField>

<ResponseField name="backgroundColor" type="string" default="transparent">
  Background color behind the dithered image.
</ResponseField>

<ResponseField name="objectFit" type="'cover' | 'contain' | 'fill' | 'none'" default="cover">
  Object fit behavior.
</ResponseField>

<ResponseField name="threshold" type="number" default="0.5">
  Threshold bias for dithering (0 to 1).
</ResponseField>

<ResponseField name="animated" type="boolean" default="false">
  Enable animation effect.
</ResponseField>

<ResponseField name="animationSpeed" type="number" default="0.02">
  Animation speed (lower = slower).
</ResponseField>

<ResponseField name="class" type="string">
  Additional CSS classes for the container (use this to set size via Tailwind).
</ResponseField>
