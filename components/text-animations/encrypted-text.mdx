---
title: "Encrypted Text"
description: "A text reveal effect that starts as scrambled characters and progressively decrypts into the final message when it enters the viewport."
icon: "lock"
---

Create a captivating decryption effect that reveals text character by character from scrambled glyphs.

## Usage

```vue
<EncryptedText
  text="Your secret message"
  :revealDelayMs="50"
  :flipDelayMs="50"
  charset="A-Za-z0-9!@#$%^&*"
/>
```

## Props

<ParamField path="text" type="string" required>
  The text to render and reveal.
</ParamField>

<ParamField path="class" type="string" default="">
  Base class applied to the wrapper element.
</ParamField>

<ParamField path="revealDelayMs" type="number" default="50">
  Delay (in ms) between revealing each next character.
</ParamField>

<ParamField path="flipDelayMs" type="number" default="50">
  Delay (in ms) between re-scrambling unrevealed characters.
</ParamField>

<ParamField path="charset" type="string" default="A-Z a-z 0-9 !@#$%^&*()_+-={}[];:,.<>/?">
  Character set used for the scrambled glyphs.
</ParamField>

<ParamField path="encryptedClass" type="string" default="">
  Class applied to characters while still encrypted/scrambled.
</ParamField>

<ParamField path="revealedClass" type="string" default="">
  Class applied to characters once revealed.
</ParamField>
