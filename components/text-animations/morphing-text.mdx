Create smooth, organic transitions between different text strings with morphing animations.

## Usage

```vue
<MorphingText
  :texts="['Hello', 'World', 'Morphing', 'Text']"
  :morphTime="1.5"
  :coolDownTime="0.5"
/>
```

## Props

<ParamField path="texts" type="string[]" default="[]">
  Array of texts to morph between.
</ParamField>

<ParamField path="class" type="string" default="">
  Additional classes for the container.
</ParamField>

<ParamField path="morphTime" type="number" default="1.5">
  Animation execution time.
</ParamField>

<ParamField path="coolDownTime" type="number" default="0.5">
  Animation dwell time.
</ParamField>
