---
title: "Container Text Flip"
description: "A container that flips through words, animating the width."
icon: "arrows-rotate"
---

Create smooth word transitions with animated container width adjustments.

## Usage

```vue
<ContainerTextFlip
  :words="['better', 'modern', 'beautiful', 'awesome']"
  :interval="3000"
  :animationDuration="700"
/>
```

## Props

<ParamField path="words" type="string[]" default="['better', 'modern', 'beautiful', 'awesome']">
  Array of words to cycle through in the animation.
</ParamField>

<ParamField path="interval" type="number" default="3000">
  Time in milliseconds between word transitions.
</ParamField>

<ParamField path="animationDuration" type="number" default="700">
  Duration of the transition animation in milliseconds.
</ParamField>

<ParamField path="class" type="string" default="">
  Additional CSS classes to apply to the container.
</ParamField>

<ParamField path="textClass" type="string" default="">
  Additional CSS classes to apply to the text.
</ParamField>
