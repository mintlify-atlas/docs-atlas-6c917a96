Add a radiant glare effect that sweeps across your text.

## Usage

```vue
<RadiantText
  :duration="10"
  :radiantWidth="100"
>
  Radiant Text
</RadiantText>
```

## Props

<ParamField path="duration" type="number" default="10">
  Duration of the animation in seconds.
</ParamField>

<ParamField path="radiantWidth" type="number" default="100">
  Width of the radiant animation.
</ParamField>
