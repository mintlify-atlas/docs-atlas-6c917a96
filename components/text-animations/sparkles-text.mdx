Highlight your text with continuous sparkle animations and customizable colors.

## Usage

```vue
<SparklesText
  text="Sparkles Text"
  :sparklesCount="10"
  :colors="{first: '#A07CFE', second: '#FE8FB5'}"
/>
```

## Props

<ParamField path="class" type="string" default="-">
  The class to be applied to the sparkles text.
</ParamField>

<ParamField path="text" type="string" default="">
  The text to display.
</ParamField>

<ParamField path="sparklesCount" type="number" default="10">
  Number of sparkles that appear on the text.
</ParamField>

<ParamField path="colors" type="object" default="{first: '#A07CFE', second: '#FE8FB5'}">
  The sparkles colors with `first` and `second` properties.
</ParamField>
