Create animated background fill effects to highlight important text.

## Usage

```vue
<TextHighlight
  :delay="0"
  :duration="2000"
  textEndColor="inherit"
>
  Highlighted text
</TextHighlight>
```

## Props

<ParamField path="delay" type="number" default="0">
  Delay before the animation starts, in `ms`.
</ParamField>

<ParamField path="duration" type="number" default="2000">
  Duration of the animation, in `ms`.
</ParamField>

<ParamField path="textEndColor" type="string" default="inherit">
  Color of the text at the end of the animation. Match WCAG recommendations.
</ParamField>
