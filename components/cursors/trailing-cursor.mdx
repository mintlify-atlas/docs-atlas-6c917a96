A stunning WebGL-powered cursor trail that creates smooth, flowing ribbons following your mouse movement. Perfect for adding a touch of visual flair to interactive experiences.

## Installation

<Steps>
  <Step title="Install dependencies">
    This component requires OGL (a lightweight WebGL framework):
    ```bash
    npm install ogl
    ```
  </Step>
  
  <Step title="Copy component code">
    Add the `TailedCursor.vue` component to your project.
  </Step>
</Steps>

## Usage

### Basic Usage

```vue
<template>
  <TailedCursor />
</template>

<script setup>
import TailedCursor from '@/components/inspira/ui/TailedCursor.vue'
</script>
```

### Custom Colors

```vue
<template>
  <TailedCursor :colors="['#ff0080', '#7928ca', '#00d4ff', '#50e3c2']" />
</template>
```

### Adjust Trail Physics

```vue
<template>
  <TailedCursor 
    :baseSpring="0.05" 
    :baseFriction="0.85"
    :speedMultiplier="0.8"
  />
</template>
```

### Enable Shader Effects

```vue
<template>
  <TailedCursor 
    :enableShaderEffect="true"
    :effectAmplitude="3"
    :enableFade="true"
  />
</template>
```

### Custom Ribbon Thickness

```vue
<template>
  <TailedCursor 
    :baseThickness="40"
    :pointCount="60"
  />
</template>
```

## Props

<ResponseField name="colors" type="string[]" default='["#ff9346", "#7cff67", "#ffee51", "#00d8ff"]'>
  Array of colors for each tail ribbon. Each color creates a separate ribbon trail.
</ResponseField>

<ResponseField name="baseSpring" type="number" default="0.03">
  Spring strength for tail movement responsiveness. Higher values make the tail follow more closely.
</ResponseField>

<ResponseField name="baseFriction" type="number" default="0.9">
  Friction factor slowing down the tail movement. Lower values create more drag.
</ResponseField>

<ResponseField name="baseThickness" type="number" default="30">
  Base thickness of the tail ribbons in pixels.
</ResponseField>

<ResponseField name="offsetFactor" type="number" default="0.05">
  Horizontal offset factor between each ribbon line. Creates separation between ribbons.
</ResponseField>

<ResponseField name="maxAge" type="number" default="500">
  Maximum age for tail segments controlling their fade timing.
</ResponseField>

<ResponseField name="pointCount" type="number" default="50">
  Number of points composing each ribbon tail. More points create smoother trails.
</ResponseField>

<ResponseField name="speedMultiplier" type="number" default="0.6">
  Speed multiplier controlling the animation speed.
</ResponseField>

<ResponseField name="enableFade" type="boolean" default="false">
  Enables fading effect on tail edges for a softer appearance.
</ResponseField>

<ResponseField name="enableShaderEffect" type="boolean" default="false">
  Enables dynamic shader wave effect on ribbons for additional visual interest.
</ResponseField>

<ResponseField name="effectAmplitude" type="number" default="2">
  Amplitude of the shader wave effect when enabled. Higher values create more pronounced waves.
</ResponseField>

<ResponseField name="backgroundColor" type="number[]" default="[0, 0, 0, 0]">
  RGBA background clear color for the WebGL canvas. Format: [red, green, blue, alpha] with values 0-1.
</ResponseField>

## Features

- **WebGL Rendering**: Hardware-accelerated smooth performance
- **Multi-colored Ribbons**: Support for multiple color trails
- **Physics-based Motion**: Natural spring and friction dynamics
- **Shader Effects**: Optional wave and fade effects
- **Highly Customizable**: Fine-tune every aspect of the animation

## Performance Notes

- Uses WebGL for GPU acceleration
- Lightweight OGL framework (smaller than Three.js)
- Optimized for real-time cursor tracking
- Adjust `pointCount` to balance quality vs performance
